# 운전면허 필기시험 🚗

한국 운전면허 필기시험 준비를 위한 모바일 퀴즈 웹앱입니다.
도로교통공단 공식 문제은행을 기반으로 학습할 수 있습니다.

## 기능

### 연습 모드 📝
- 15문제 랜덤 출제
- 즉시 정답/오답 피드백
- 스트릭 보너스 & XP 시스템
- 실시간 합격률 추적 (60% 기준)

### 모의시험 모드 🎯
- 40문제 · 50분 타이머
- 실제 시험과 동일한 환경
- 60점 이상 합격/불합격 판정
- 문제 번호 네비게이션으로 자유롭게 이동

### 결과 화면
- 애니메이션 점수 원형 차트
- 합격 시 축하 컨페티 효과
- 정답 수, XP, 최고 스트릭 통계

## 기술 스택

- **Next.js 16** — Static Export (`output: 'export'`)
- **TypeScript**
- **Tailwind CSS v4**
- **Motion** (motion.dev) — 화면 전환, 피드백 애니메이션
- **GitHub Pages** — 정적 배포

## 시작하기

```bash
npm install
npm run dev
```

빌드 및 정적 내보내기:

```bash
npm run build
npx serve out
```

## 배포

`main` 브랜치에 푸시하면 GitHub Actions가 자동으로 GitHub Pages에 배포합니다.
